================================================================================
                        DATABASE SCHEMA
================================================================================

TABLES:
  • books (11,306 records) - Public book catalog
  • user_books - User-tracked books with ratings/notes
  • shelves - User's custom shelves
  • shelf_books - Books in each shelf
  • profiles - User profile information

RELATIONSHIPS:
  • 1 User → Many Books (tracked)
  • 1 User → Many Shelves
  • 1 Shelf → Many Books
  • Books have no ownership (public read-only)

SECURITY:
  • RLS policies enforce data isolation
  • Books table: Public read-only
  • User tables: User-only access
  • Sessions stored in HTTP-only cookies
